# Vue 3 Project Structure Template

A clean and scalable Vue 3 starter structure designed for enterprise-level front-end applications.

## 🔧 Technologies Used

- **Vue 3** – Progressive JavaScript Framework
- **Node.js v22** – Runtime environment for API/back-end integration
- **Tailwind CSS v3** – Utility-first CSS framework for modern UI design
- **DaisyUI v4** – Tailwind CSS component library
- **Pinia** – Vue’s official state management library
- **SweetAlert2** – Beautiful and responsive alert/dialog library
- **js-cookie** – Lightweight cookie handling library
- **Axios** – Promise-based HTTP client

## 📁 Project Structure

```bash
Vue-Structure/
├── public/              # Public assets
├── src/
│   ├── assets/          # Images, fonts, and static files
│   ├── components/      # Reusable Vue components
│   ├── layouts/         # Layout templates
│   ├── pages/           # Route-based views
│   ├── router/          # Vue Router configuration
│   ├── stores/          # Pinia store modules
│   ├── App.vue          # Main App component
│   └── main.js          # App entry point
├── .env                 # Environment variables
├── tailwind.config.js   # Tailwind CSS configuration
├── package.json         # Project metadata and dependencies
└── README.md            # Project documentation
